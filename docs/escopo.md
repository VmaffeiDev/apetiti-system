1. ESCOPO DO SOFTWARE — APETITI (Telegram Bot)
🎯 Objetivo

Desenvolver um sistema de recomendação de refeições via Telegram, capaz de interagir com o usuário por meio de diálogo contínuo, sugerindo opções de acordo com o contexto (almoço ou jantar) e aprendendo com as preferências do usuário.

📌 Escopo

O sistema Apetiti consiste em um chatbot integrado ao Telegram que:

Interage com o usuário via mensagens
Identifica o contexto da refeição (almoço ou jantar)
Sugere pratos com base no contexto
Permite feedback do usuário (👍 ou 👎)
Aprende preferências básicas do usuário
Mantém estado de conversa (diálogo contínuo)

🚫 Fora do escopo (Sprint 01)
Integração com banco de dados real
Integração com restaurantes reais
Pagamento ou pedidos
IA avançada (machine learning real)
👤 Público-alvo
Usuários de restaurantes/self-service
Pessoas indecisas sobre o que comer
